#### Requisitos

1. Conexão com MongoDB
2. Rota POST para inserir dados no DB (inclusive imagens)
3. Rota GET para exibir os dados armazenados no DB
4. Rota PUT para atualizar dados no DB
5. Rota DELE para excluir dados do DB
